ygettime(3) - ytool man page

Name

ygettime - 시스템 경과시간

Synopsis

#include "libytool.h"

time_t ygettime(struct timespec *ts);

Description

이 함수는 시스템 부팅 후 경과시간을 초(sec)단위로 리턴한다. ts는 NULL로 호출할 수 있다. 또한 유효한 timespec 구조체의 포인터이면 경과시간을 timespec에 맞게 기록한다.

ts
time.h에 정의되어 있는 timespec 구조체의 포인터 변수이다.
struct timespec {
	time_t   tv_sec;        /* seconds */
	long     tv_nsec;       /* nanoseconds */
};

Return Value

성공 시 시스템의 부팅 후 경과시간을 초단위로 리턴한다. 실패 시 (time_t)-1 의 값을 리턴한다.

Notes

ts는 NULL일 수 있다.

Bugs

Example

아래 프로그램의 실행 결과는 다음과 같다.

$ ./a.out
20208:78
20208.78

Program source

#include <stdio.h>
#include "../libytool.h"

int main(void)
{
    struct timespec ts;
    float elapsed;

    ygettime(&ts);
    yfcat("/proc/uptime", "%f", &elapsed);
    printf("%lu:%ld\n%.2f\n", ts.tv_sec, ts.tv_nsec / 10000000, elapsed);

    return 0;
}
