yexecl(3) - ytool man page

Name

yexecl - 형식화된 Command Line을 실행

Synopsis

#include "libytool.h"

int yexecl(char *pathname, const char *arg, ...);

Description

이 함수는 자식 프로세스를 fork(2)한 후 지정된 파일을 실행한다. 실행 파일이 종료될 때 까지 waitpid(2)로 기다린다. 형식화된(Formatted) Command Line을 생성한 후 yexecv에서 정의된 규칙에 의해 명령줄을 Parsing 한다. 그리고 첫 번째 인자로 지정된 파일을 실행한다.

pathname
NULL이면 표준 입출력은 부모로 부터 상속된다. 또는 Redirection 지시자('>')와 파일 또는 디바이스명의 조합으로 전달할 수 있다.
arg
형식화된 Command Line을 넘겨준다.

Return Value

실행 파일의 종료 코드를 리턴한다. 또는 errno 일 수 있다.

Notes

단일 Job을 실행한다. 즉 Pipe로 연결하여 여러 Job을 동시에 실행할 수는 없다.

Bugs

Example

아래 프로그램의 실행 결과는 다음과 같다.

$ ./a.out
Chain INPUT (policy ACCEPT 13063 packets, 1997K bytes)
 pkts bytes target     prot opt in     out     source               destination

Hello World!

Program source

#include <stdio.h>
#include "../libytool.h"

int main(void)
{
	yexecl(NULL, "iptables -t %s -nvL INPUT", "filter");

	yexecl(">/dev/null 2>&1", "iptables -t %s -F MINIUPNPD", "nat");

	yexecl(NULL, "sh -c \"echo Hello World!\"");

	return 0;
}
